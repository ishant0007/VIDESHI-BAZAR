* Students studying about stock market

# USAGE - 
The script is designed to be user-friendly and easy to use.

Environment required - Linux

Langauge required to be installed - python 3

Step 1 - download zip file of project and extract

Step 2 - install requirements.txt with pip

Step 3 - (in-terminal) apt-get install boxes

Step 4 - (in-terminal) chmod +x bazaar.sh

Step 5 - (in-terminal) ./bazaar.sh

Step 6 - Enter the argument (stock) you want the data of.

# example - AMZN 2022-10-20 2022-11-20

{stock tikcer} + {start date} + {end date}.

# That's all, data and charts will be generated automatically.
